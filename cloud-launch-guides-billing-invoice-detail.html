Cloud Launch Guides - Billing Invoice Detail

Invoicing is often an after-the-fact part of the Getting Started process. It is an important part nonetheless. In this screencast, I show how to find a summary of charges. Then we download the raw data, filter and calculate the information to answer a specific request.

Key takeaways

  • Billing Summary can be found in the Control Panel username drop-down by clicking on "Billing and Payments."
  • Exporting the invoice .csv file provides a more detailed inspection of charges.
  • Using almost any spreadsheet program, you can filter and calculate the charges for specific services.
